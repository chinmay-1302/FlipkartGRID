## Flipkart GRID 5.0 - Robotics Challenge
This repo is to maintain the DL models & python application of our team for the Flipkart GRID 5.0 - Robotics Challenge

### Tech Stack
- OpenCV
- YOLOv8
- Roboflow

### Models
- Box Detection Model
- Barcode Detection Model
